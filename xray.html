7 个 Agent · 各管一件事

Q-Agent · 解析问题意图
Heart-Agent · 算心动浓度数字
Verdict-Agent · 写锐利判词
T-Agent · 以 TA 口吻生成回话
P-Agent · 把回话精炼到符合人设
Diary-Agent · 写一段"没说出口的话"
Diagnosis-Agent · 综合多段做判读
为什么拆成 7 个 Agent 而不是一个大 prompt: 职责单一 → 每个 Agent 输出更小更稳、可并发省延迟、可独立 A/B 与 eval。可靠性不是拍脑袋:离线 eval 实测 28/28 调用返回合法结构 →
可验证算分: 契合分走确定性算法、明细 100% 可验证;心动浓度等 LLM 输出不确定,改用行为单调性断言盯回归(实测 4/4 通过)。
